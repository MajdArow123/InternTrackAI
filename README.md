# InternTrackAI

InternTrackAI is an ASP.NET Core MVC web application for tracking internship and job applications.

## Features
- Add job applications
- Track company name, role title, job link, location, work mode, and status
- Store deadline, date applied, salary, and job description
- Built with ASP.NET Core MVC, Entity Framework Core, SQLite, and Identity

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- ASP.NET Core Identity
